# HivePanel

HivePanel is a free, open-source game server management platform built with Laravel, Vue, and Go. Designed to be fast, modern, and developer-friendly, HivePanel runs game servers through isolated runtimes while providing a clean, intuitive interface for administrators and players alike.

Powered by a modular ecosystem of Panels, Workers, Registry, and Combs, HivePanel makes deploying, managing, and scaling game servers simple—from a single node to large distributed fleets.
